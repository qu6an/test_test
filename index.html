Getting Started with H5P

This starter kit provides the basic structure for implementing H5P content in your web application.

Follow these steps to get started:

  1. Add your H5P content files to the h5p/content directory
  2. Install H5P libraries in the h5p/libraries directory
  3. Update the H5PIntegration object with your content configuration
  4. Load content using the H5P API
